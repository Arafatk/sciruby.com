The SciRuby Manifesto

Ruby has for some time had no equivalent to the beautifully constructed NumPy, SciPy, and matplotlib libraries for Python.

We believe that the time for a Ruby science and visualization package has come and gone. Sometimes when a solution of sugar and water becomes super-saturated, from it precipitates a pure, delicious, and diabetes-inducing crystal of sweetness, induced by no more than the tap of a finger. So it is, we believe, with the need for numeric and visualization libraries in Ruby.

We are not the first with this idea, but we are trying to bring it to life.

Project Status

At this point, SciRuby is not a single gem (there is a sciruby gem, but it's really just a placeholder). If you want to use the numerical, statistical, scientific, and visualization tools that have been written thus far, we recommend you try out the following component gems:

  • Visualization: rubyvis
  • Statistics: statsample
  • Numeric: narray, minimization, integration
  • Probability: distribution

Please note that we are in the process of rewriting NArray, and that none of these are finished products. If you like what you see, or would like to get involved, please join our Google Group (see below) or find us in #sciruby on IRC (Freenode.net).

Getting SciRuby

Word to the wise: These gems have been tested, but not extensively. If you're thinking of using SciRuby to write mission critical code, such as for driving a car or flying a space shuttle, you may wish to choose other software (for now).

gem install rubyvis statsample minimization distribution integration

Note that distribution and integration can optionally make use of gsl and statistics2, which implement C versions of several of the methods. They may speed things up a bit, and have been tested much more thoroughly. These can be installed like so:

gem install gsl statistics2
